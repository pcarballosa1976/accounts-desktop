En esta carpeta deberá encontrar los fuentes del programa de 32 bit para la plataforma Windows denominado Accounts, se trata de un pequeño software elaborado enteramente con RapidQ y gratuito para todo uso.

El programa está destinado a llevar de una manera sencilla los ingresos y gastos en un hogar o en un negocio por cuenta propia donde no sea necesario (u obligatorio) utilizar un software de contabilidad más completo.
