En esta carpeta deberá encontrar los fuentes del programa de escritorio de 32 bit para la plataforma Windows denominado Accounts; se trata de un pequeño software elaborado enteramente con RapidQ y gratuito para todo uso.

El programa está destinado a llevar de una manera sencilla los ingresos y gastos en un hogar o en un negocio por cuenta propia donde no sea necesario (u obligatorio) utilizar un software de contabilidad más completo.

Es libre de modificar este programa como lo crea conveniente (según los términos de la licencia GNU-3.0) para hacerlo más adecuado para su caso particular o incluirle más funcionalidades.

En caso de necesidad, o si tiene una idea para mejorar este programa o para desarrollar otro y no sabe cómo llevar a cabo su idea, también puede comunicarse con su autor.

El blog del autor está en la dirección https://cubansolutions.blogspot.com donde también podrá encontrar y descargar gratis el binario de este programa si no desea compilarlo.

El software puede ser compilado utilizando el entorno visual de RapidQ (RapidQ.exe) y abriendo y corriendo Accounts.bas, o si le parece mejor también puede hacerlo desde la línea de comandos con el comando RC Accounts.bas (recuerde colocarse en la carpeta de RapidQ y poner la ruta hasta la carpeta donde tiene Accounts.bas y el resto de los archivos del programa).
